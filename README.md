# LowissShop - Frontend

La parte de delante de LowissShop, hecha con React + Vite.

### Qué hace

Web para ver productos, registrarte, loguearte, subir cosas, ver tus anuncios, favoritos y tal.

### Pantallas principales

- Inicio → lista de productos
- Registro → crear cuenta (con foto opcional)
- Login → entrar
- Dashboard → tus productos (aún básico)

### Cómo ejecutarlo

1. Clona el repo
2. `npm install`
3. Crea `.env` con:
VITE_API_URL=http://localhost:3000/api
text4. `npm run dev`

Abre http://localhost:5173
